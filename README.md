## Introduction

SpectraPlay is a sleek, interactive web application built to deliver a rich media‐playing experience with smooth UI, clean animations, and optimized performance.
Live Demo: **[https://himeshdua.github.io/SpectraPlay](https://himeshdua.github.io/SpectraPlay)**

This project serves as a showcase of modern front-end engineering and user-centric media playback design.

## Features

* **Dynamic media playback**: Handle audio/video streaming with custom controls
* **Responsive design**: Works fluidly across mobile, tablet, and desktop
* **Animated UI transitions**: Smooth page and element animations for an immersive feel
* **Dark/light theme support**: Built-in readiness for multiple themes
* **Modular component architecture**: Reusable UI blocks and clean folder structure
* **Accessibility-focused**: Keyboard controls, ARIA labels, semantic HTML
* **Performance optimised**: Lazy-loaded assets, optimized bundle size

## Installation

Clone this repository:

```bash
git clone https://github.com/HimeshDua/SpectraPlay.git  
cd SpectraPlay  
```

Install dependencies:

```bash
npm install  
```

Run locally:

```bash
npm run dev  
```

Then go to `http://localhost:3000`


## Deployment

The project is configured for deployment via **GitHub Pages**, but you can use any static-site hosting.

To deploy:

```bash
npm run build  
npm run deploy  
```

(Or set up GitHub Actions to auto-deploy from the `main` branch.)
